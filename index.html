Aucbot-Discord's auction bot

How to use it?

To set the logging channel, use the `[p]slc [channel]` command. Replace `[channel]` with either the direct mention or the ID of the channel. Now you ready to get going with sell command. `/Sell` command takes player(that want to sell), baseprice and increment as perimeters. To clarify, both the "baseprice" and "increment" parameters can take the following format: - m for millions (e.g. 4m for 4 million) - b for billions - t for trillions - k for thousands However, it cannot be written as "4 millions”. You are now ready to place bids. You can bid using the `[p]bid` command, but you cannot bid manually, for example, by typing [p]bid 4m. All bids must be incremented. To close out the bidding process, run the `[p]sold` command. This will add the player to the last bidder's squad and deduct the amount of the last bid.
